# Cli

Package with utilities to interact with the command line:
* Output interactions
* Input interactions
* Running processes

## Installation

```shell
pip install quinten-cli
```

## Usage
Use 

```shell
import cli
```
instead of

```shell
import subprocess
```

to run processes
